# find-the-film-rating

<img src="https://travis-ci.org/kgentner/find-the-film-rating.svg?branch=master"></img>

### Summary
A simple web application that returns the MPAA rating for a movie that a user inputs.


### Technologies

#### Main (Web)
* Node.js
* Express.js
* Grunt
* Browserify

#### Testing & Linting
* Mocha
* Chai
* JSHint
* JSCS
